Get started

We will see in this section how to install and run your "hello world" using Gin.

Requirementsâ€‹

  • Go: any one of the three latest major releases

You might try to run Gin on other versions of Go, it might work, but nothing guarantees that everything will function as planned.

Gin Instalationâ€‹

  1. Initialize Your Go Module: In your project directory, initialize a new Go module by running the command:
go mod init <module-name>
  1. Install Gin: To install Gin, use the go get command:
go get -u github.com/gin-gonic/gin
  1. Import Gin in Your Project: In your Go files, import the Gin package to start using it:
import "github.com/gin-gonic/gin"
  1. Done: You and your project are ready to use Golang!

First Applicationâ€‹

Now that you have installed and imported Golang into your project, you can create your first application.

  1. Create a Gin Router: Create a new Gin router in your main Go file to handle HTTP requests:
r := gin.Default()
  1. Define Routes: Define routes and associate them with handler functions. For example:
r.GET("/init", func(c *gin.Context) {
  c.JSON(200, gin.H{
    "message": "hello world!",
  })
})
  1. Run Your Server: Finally, start your server with the Run method:
r.Run()

// By default, it runs on http://localhost:8080
// You can change port using: r.Run(":3000")
  1. Result: At this moment, your code should look something like this:
package main

import "github.com/gin-gonic/gin"

func main() {
  r := gin.Default()

  r.GET("/init", func(c *gin.Context) {
    c.JSON(200, gin.H{
      "message": "hello world!",
    })
  })

  r.Run()
}
  1. Run code: You can run the code with go run main.go:
go run main.go
  1. Test Your Application: Test your application by accessing the defined routes in a web browser or using a tool like curl:
curl http://localhost:8080/init
  1. Congratulations: You have created your first application in Go!
